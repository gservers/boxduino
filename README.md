## BOXmodUINO - the first Arduino powered e-cigarette!

Using atMega328P (Arduino Nano) to control 3034 N-channel MOSFET transistor with pwm.

Version 1.0:
  * HD77480 display
  * Sleep / Power on
  * Firing
  * Read input voltage
  * Very simple control
  * Battery indicator
  
Check Issues for ToDo list.
