BOXmodUINO - one of the first Arduino powered e-cigarettes!

Using atMega328P (Arduino Nano) to control 3034 N-channel MOSFET transistor with pwm.

Version 1.0:
  Working:
    -HD77480 display\n
    -Sleep / Power on\n
    -Firing\n
    -Read input voltage\n
    -Very simple control\n
